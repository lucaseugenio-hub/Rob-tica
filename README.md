# Sobre o código
- Esse arquivo .ino que eu mandei é o código que eu estava testando os sensores de refletância (os quais eu fiz um modelo na lousa)
- Provavelmente ele não está 100% funcionando, então quero que vocês entendam o funcionamento teórico do sensor e tentem faze-lo funcionar juntamente aos motores.
- Para que não fique muita coisa, utilizem apenas dois sensores, os dois mais próximo do centro, façam com que se os sensores identificarem branco as rodam andem para frente e se identificarem preto andem para traz.
